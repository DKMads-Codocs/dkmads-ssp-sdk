# Unity quickstart sample

Attach `QuickstartSample` to a GameObject. Set **integration key** and ad unit UUIDs in the Inspector.

Context menu actions (right-click component):

1. Load and show interstitial  
2. Load and show app open (splash ad unit)  
3. Present Ad Inspector  

Requires native SDK install per [Unity integration guide](../../../docs/integration/unity.md).
